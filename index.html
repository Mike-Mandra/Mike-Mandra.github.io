Choosing a programming language like Python is a great way to start. Enroll in post-secondary education for computer science or computer programming and hone your skills. Build projects you're interested in, such as your own web server or a finance application. Participate in online communities on Discord or Reddit to talk and work with like-minded people. Stay up to date on the latest tech news using sites like TechCrunch or TheVerge. Learn a stack by learning backend languages like Python, Java, and C# while also learning front-end languages like Javascript, HTML, and CSS. Choose if you want to be a web developer, data scientist, software engineer, or software developer and what languages you'd like to use. Complete your degree and start applying to jobs!
